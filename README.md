# Numer0n
C言語でNumer0nを作ってみた。

対戦型ではなく、ランダムに出される数字を当てるだけです。
さすがにヒントがないときついなということで、スキルを３つだけ用意しました。
さらっと書いてあるので察してください。

コマンドプロンプト上でコンパイル、実行して下さい。

=========================

gcc -o Numer0n Numer0n.c
./Numer0n
==========================
